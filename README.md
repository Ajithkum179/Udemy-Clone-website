🎓 Udemy Clone Website

A responsive Udemy Clone built using HTML5 and CSS3, designed to replicate the core layout and user interface of the Udemy website. This project focuses on front-end design, layout structuring, and responsive web development principles.

📌 Project Description

This project is a static clone of the Udemy homepage created for learning and practice purposes. It showcases modern UI components such as navigation bars, course cards, categories, banners, and footers while maintaining a clean and user-friendly design.

The goal of this project is to strengthen foundational web development skills using only HTML and CSS, without JavaScript or external frameworks.
